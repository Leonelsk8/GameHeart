Info

La tematíca de esta pagina es sobre noticias de estrenos de videojuegos. Hecha con HTML CSS JAVASCRIPT y BOOTSTRAP.
Usando JSON-SERVER como base de datos ficticio, donde los datos se guardan en "db.json", para poder interactuar con esta base de datos es necesario tener instalado "node.js" e iniciar el servidor con un script en una terminal desde el codigo, escribiendo solamente "npm start scripts" en la terminal de visual studio code, una vez iniciado ya se podrá registrar e iniciar sesión.
